## 🔗 2. Sub-Skills Overview

Logical system layers derived:

1. **Compute Processing (CPU Basics)**
   → Executes instructions and logic

2. **Memory Management (RAM Usage)**
   → Stores active data for fast access

3. **Storage Layer (HDD vs SSD)**
   → Persistent data storage

4. **Server Infrastructure (Rack Servers)**
   → Hosts applications and services

5. **Network Routing (Router Basics)**
   → Directs traffic across networks

6. **Network Switching (Switch Basics)**
   → Routes data within local networks

   → Filters and secures traffic
