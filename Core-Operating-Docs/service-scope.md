# Service Scope Framework

## Core service category
Small-business IT systems and workflow solutions

## Included work types
- Business process review
- Tool setup and organization
- Spreadsheet and tracker systems
- Documentation and checklists
- Basic automation guidance
- Simple digital presence support

## Not included by default
- Large custom software development
- 24x7 support
- enterprise infrastructure management
- legal or financial advisory work
- unmanaged change requests outside scope

## Delivery model
- Discovery first
- Fixed-scope project where possible
- Remote-first execution
- Documentation-backed handover
- Optional support after completion
