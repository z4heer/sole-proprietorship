# Client Delivery Checklist

## Before starting
- Client intake form received
- Requirement checklist completed
- Proposal approved
- Payment terms confirmed
- Project folder created

## During project
- Discovery completed
- Scope confirmed in writing
- Work plan prepared
- Client updates shared
- Testing done internally

## Before handover
- Deliverables completed
- Final review done
- Client walkthrough ready
- Documentation prepared
- Pending issues listed

## Handover
- Files delivered
- Access handed over
- Usage instructions shared
- Final approval requested
- Final invoice sent if applicable

## After delivery
- Feedback requested
- Testimonial requested
- Support options shared
- Project archived
