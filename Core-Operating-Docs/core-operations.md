Here’s how the **10 core operations** break into **sub-systems** with specific **activities** for a solo founder. Each sub-system maps to a dedicated section in your Notion dashboard, Google folder, or Sheet, with repeatable activities tied to daily/weekly rhythms. [notion](https://www.notion.com/templates/personal-dashboard-2026)

## 01_Business_Admin
**Sub-systems:** Legal, Finance Docs, Compliance.

| Sub-system | Activities |
|------------|------------|
| Legal | File registrations; draft/review contracts; sign NDAs; archive approvals. |
| Finance Docs | Generate/send invoices; track payments; log receipts; prepare tax bundles. |
| Compliance | Update licenses; file GST/returns; backup docs monthly. |

## 02_Brand
**Sub-systems:** Identity, Messaging, Assets.

| Sub-system | Assets |
|------------|------------|
| Identity | Brainstorm names; design logo/colors; register domain/trademarks. |
| Messaging | Write bios; craft taglines; update profiles (LinkedIn, website). |
| Assets | Create icons; mood boards; brand guidelines doc. |

## 03_IT_Services
**Sub-systems:** Proposals, Delivery, Templates.

| Sub-system | Activities |
|------------|------------|
| Proposals | Customize offers; send quotes; negotiate scopes; sign contracts. |
| Delivery | Build/test/deploy; client demos; handover docs; invoice. |
| Templates | Update SOPs; refine proposals; create reusable code snippets. |

## 04_Trading_Products
**Sub-systems:** Ideas, Development, Tracking.

| Sub-system | Activities |
|------------|------------|
| Ideas | Sketch app features; research markets; prioritize backlog. |
| Development | Code prototypes; test logic; screenshot progress. |
| Tracking | Update journal; log trades; analyze performance. |

## 05_Courses
**Sub-systems:** Planning, Creation, Delivery.

| Sub-system | Activities |
|------------|------------|
| Planning | Outline modules; define audience; set pricing. |
| Creation | Write scripts; design slides; record videos; make worksheets. |
| Delivery | Host on platform; promote; gather feedback; update. |

## 06_Showcase_Website
**Sub-systems:** Portfolio, Demos, Social Proof.

| Sub-system | Activities |
|------------|------------|
| Portfolio | Build case studies; upload demos; optimize site. |
| Demos | Record videos; create interactive prototypes; test links. |
| Social Proof | Collect testimonials; display reviews; update metrics. |

## 07_Content_Marketing
**Sub-systems:** Planning, Creation, Distribution.

| Sub-system | Activities |
|------------|------------|
| Planning | Brainstorm ideas; build calendar; research keywords. |
| Creation | Draft posts/blogs; design visuals; script videos. |
| Distribution | Schedule LinkedIn/Twitter; engage comments; track views. |

## 08_Sales_CRM
**Sub-systems:** Leads, Outreach, Pipeline.

| Sub-system | Activities |
|------------|------------|
| Leads | Capture inquiries; qualify prospects; add to CRM. |
| Outreach | Send cold emails; book calls; follow up. |
| Pipeline | Update stages; forecast revenue; close deals. |

## 09_Operations
**Sub-systems:** Tasks, Quality, Workflow.

| Sub-system | Activities |
|------------|------------|
| Tasks | Prioritize daily list; time-block work; mark complete. |
| Quality | Run QA checklists; client feedback loops; iterate. |
| Workflow | Document SOPs; automate repeats; review bottlenecks. |

## 10_Finance
**Sub-systems:** Income, Expenses, Reporting.

| Sub-system | Activities |
|------------|------------|
| Income | Log payments; chase overdue; forecast cashflow. |
| Expenses | Track bills; categorize spends; cut waste. |
| Reporting | Reconcile monthly; calculate profit; plan taxes. |
