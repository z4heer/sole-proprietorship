## Notion workspace
Create one **master dashboard page** in Notion Free with these embedded databases (use free templates from Notion’s gallery): [notion](https://www.notion.com/templates/category/free-personal-planner-templates)

### 1. Tasks database
- Properties: Task, Status (To Do/In Progress/Done), Priority, Due Date, Bucket (links to your 10 areas).
- View: Kanban board by status.

### 2. Clients/Leads database
- Properties: Name, Email, Phone, Status (Lead/Active/Inactive), Next Action, Last Contact, Notes.
- View: Table sorted by next action.

### 3. Content calendar
- Properties: Post Title, Platform, Status, Publish Date, Link/Draft.
- View: Calendar by publish date.

### 4. Finance tracker (mirror Google Sheet)
- Properties: Date, Type (Income/Expense), Amount, Category, Receipt Link.
- View: Gallery with totals formula.

Link everything to the dashboard with buttons or linked views. Daily routine: Open dashboard, mark tasks, log leads. [notion](https://www.notion.com/templates/personal-dashboard-2026)

## Offline notebook routine
Use a simple paper notebook (A5 size) for zero-distraction work:

- **Morning page:** 3 priorities, 3 wins from yesterday.
- **Call/Meeting notes:** Bullet points per client.
- **Idea capture:** Quick sketches for app features, logos.
- **Weekly review:** Transfer unfinished tasks to Notion.

Scan notebook pages to Drive weekly via phone camera. [e-resident.gov](https://www.e-resident.gov.ee/blog/posts/top-digital-tools-for-solopreneurs-and-delegation-tips/)

## Daily/weekly routines
- **Daily (10 min):** Update dashboard, log expenses in Sheet, capture offline notes.
- **Weekly (30 min):** Reconcile finance Sheet, archive done tasks, plan content.
- **Monthly (1 hr):** Backup Drive to laptop, review subscriptions, export CRM data if using Zoho.

This runs 100% free, syncs across devices, and keeps you organized without app overload. [londondailyfeed](https://londondailyfeed.com/here-are-the-best-free-business-tools-for-startups/)
