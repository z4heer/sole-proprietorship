# 🧠 1. Core Understanding (WH Framework)

### What

Version Control (Git) is a system to **track changes in code, collaborate with others, and manage versions safely**.

---

### Why

To:

* Track history of changes
* Collaborate with teams
* Prevent code loss
* Manage multiple versions

---

### How

* Code stored in **repository**
* Changes saved as **commits**
* Work split using **branches**
* Combined via **merge/rebase**
* Shared using **push/pull**

---

### When

* During development
* Team collaboration
* Code deployment

---

### Where

* Local system
* Remote servers (GitHub/GitLab)
* CI/CD pipelines

---

### Who

* Developers
* DevOps engineers
* Teams

---

# 🎤 2. Explain for Fluency

### 1 Line

Git tracks code changes and helps teams collaborate.

---

### 30 Seconds

Git is a version control system where code is stored in repositories. Developers make commits, create branches for features, and merge them back. It supports collaboration using pull, push, and code reviews.

---

### 2 Minutes

Git manages code history using repositories. Developers clone repositories, make changes, and commit them. Branches allow parallel development, and merging integrates changes. Rebase helps maintain clean history. Teams collaborate using pull requests and code reviews. Conflicts may occur during merges, which must be resolved manually. Git integrates with CI/CD pipelines for automated testing and deployment, and access control ensures security.

# 📖 4. Story-Based Encoding

> You enter a project workshop.

The project is stored in a **folder (repository)**.
Every change is saved as a **checkpoint (commit)**.

Developers work on separate paths (**branches**).
These paths are later combined (**merge/rebase**).

Changes are shared using **push/pull**.
New developers copy project (**clone**).

Sometimes paths clash (**merge conflict**) and must be resolved.

Supervisors review work (**code review**).
Every version is labeled (**tag**).

Automated systems test changes (**CI trigger**).
Access is controlled (**permissions**).
