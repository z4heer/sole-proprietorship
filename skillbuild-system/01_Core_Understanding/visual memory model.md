# 📖 2. Story-Based Encoding (Narrative Memory)

👉 **Rule:** *All concepts must be part of one continuous story*

### How to apply:

* Connect all images into a **flowing story**

### Example Story:

> Workers use desk (RAM) →
> Goods come from warehouse (Storage) →
> Extra work goes to remote factory (Cloud) via roads (Network)”
