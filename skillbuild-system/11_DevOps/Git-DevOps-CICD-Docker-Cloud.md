# 📘 **1. Skill: Git Basics**

## 1. 🎯 One-Line Definition

Git is a version control system to track code changes.

## 2. 🧭 Purpose

* Collaboration
* History tracking

## 3. 🧱 Core Concept

Git = **Snapshots of code**

**Analogy:**
Save versions of document

## 4. ⚙️ Internal Working

1. Modify code
2. Stage changes
3. Commit snapshot

## 5. 🔄 System Flow

Code → Add → Commit → Repository

## 6. 🔗 Connected Systems

* CI/CD
* Teams

## 7. 💻 Practical Execution

* Messy history

## 9. 🧪 Debugging Mindset
