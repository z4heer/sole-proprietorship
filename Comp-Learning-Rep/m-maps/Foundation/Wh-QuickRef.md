# Hardware

## What
- Physical foundation of computing
- Includes servers, laptops, storage, network devices
- Executes digital operations

## Why
- Provide compute power
- Store and transfer data
- Enable software execution
- Connect users and systems

## How
- Power activates components
- CPU processes instructions
- Memory stores temporary data
- Storage keeps persistent data
- Network devices exchange information

## Core Components
- CPU
- RAM
- Storage (SSD/HDD)
- Motherboard
- Network Interface Card (NIC)
- Power Supply
- Cooling System

## Real-world Analogy
- Factory infrastructure
- Building = motherboard
- Workers = CPU
- Workspace = RAM
- Warehouse = storage
- Roads = network

## Real-world Usage
- Datacenters
- Cloud infrastructure
- Enterprise servers
- Banking systems
- Personal computing
- IoT and edge devices

## Owner
- Hardware engineers
- Infrastructure teams
- Datacenter operations
- Platform engineering

## Consumer
- Operating systems
- Applications
- Cloud platforms
- Developers
- End users

# Operating System

## What
- Core system software
- Interface between hardware and applications
- Controls system resources

## Why
- Simplify hardware usage
- Manage processes and memory
- Provide security and stability
- Enable multitasking

## How
- Receives application requests
- Allocates CPU and memory
- Manages files and devices
- Handles networking and security

## Core Components
- Kernel
- Process Manager
- Memory Manager
- File System
- Device Drivers
- Network Stack
- Shell/User Interface

## Real-world Analogy

- Open-source operating system
- Unix-like platform
- Dominant server and cloud OS

## Why
- Provide stable and scalable infrastructure
- Support automation and cloud workloads
- Enable secure enterprise operations
- Reduce licensing cost

- Linux kernel manages hardware
- Shell executes commands/scripts
- Services run in user space
- Packages provide software ecosystem

## Core Components
- Linux Kernel
- Shell (Bash)
- File System
- Systemd/Services
- Package Manager

- Security Modules
