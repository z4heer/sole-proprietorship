# Authentication & Security Basics

## Authentication

### Authentication kya hai

### Working Steps

* Banking applications

### Common Tools

* OAuth
* LDAP

### Troubleshooting Thinking

* Wrong credentials?

### Communication Practice

* “Authentication user identity verify karta hai.”
* “Modern systems token-based authentication use karte hain.”
