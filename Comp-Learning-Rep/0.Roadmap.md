# 🧪 STEP 4 — HOW TO LEARN DEEPLY

---

# ❌ Avoid

* passive reading
* tutorial addiction
* memorization

---

# ✅ Instead Use

## 1. Visual Flow Thinking

Always visualize:

```text id="sk4zyq"

```

## 2. Speak Aloud

Very important.

## 3. Build Tiny Projects

Even very small.

* login API
* file upload
* dockerized app

This develops engineering thinking.

Ask:

* what breaks?
* scaling issue?
* latency?
* retry?
* security risk?
