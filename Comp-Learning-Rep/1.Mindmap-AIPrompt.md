# 1. MASTER POSITIONING MIND MAP PROMPT

This is the MOST IMPORTANT map.

Use first for every technology.

---

## Prompt

```text id="q5kpyv"
Act as a senior engineer and solution architect.

Create a MASTER POSITIONING MIND MAP for [SKILL].

Cover:

1. What it is
2. Why it exists
3. What problems it solves
4. Where it fits in enterprise architecture
5. Which layer it belongs to
6. Who uses it
7. Core features
8. Inputs and outputs
9. Dependencies
10. Alternatives
11. Architecture impact
12. Scaling impact
13. Security impact
14. Monitoring metrics
15. Common production issues
16. Real-world usage
17. How senior engineers think about it
18. How solution architects think about it

Format:
- hierarchical tree structure
- visual thinking style
- simple beginner-to-advanced explanation
- enterprise-oriented
```
