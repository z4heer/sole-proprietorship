# Your Mental Framework (VERY IMPORTANT)

For every system ask:

## 1. WHAT

What is the system?

## 2. WHY

Why does it exist?

## 3. WHO

Who are actors/services/users?

## 4. HOW

How does request move?

## 5. WHERE

Where is data stored?

## 6. WHEN

When are queues/events triggered?

## 7. WHAT IF

What can fail?

## 8. SCALE

How does it scale?

This framework alone can transform your thinking.
