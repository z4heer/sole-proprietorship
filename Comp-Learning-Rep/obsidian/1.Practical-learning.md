##
* Linux Kernel is the core component of Linux. It is used to manage hardware, memory, and processes. It works by acting as a bridge between applications and hardware.

* Operating System is system software that manages computer resources. It is used to run applications and control hardware. It works by coordinating memory, CPU, storage, and devices.

* Linux Distribution is a complete operating system built on the Linux kernel. It is used for desktop, server, or security purposes. It works by combining the kernel with software tools and package managers.

* Open Source is software with publicly available source code. It is used to allow modification and collaboration. It works by letting developers access and improve the code freely.

* Terminal is a text-based interface for interacting with Linux. It is used to execute commands and manage systems. It works by sending typed commands to the shell.

* Shell is a command interpreter in Linux. It is used to process user commands. It works by converting commands into system-level operations.

* Bash is a Linux shell and scripting environment. It is used for command execution and automation. It works by interpreting shell commands and scripts.

* Virtualization is technology for running multiple operating systems on one machine. It is used to create isolated environments. It works by abstracting physical hardware into virtual resources.

* Hypervisor is software that manages virtual machines. It is used to allocate hardware resources to VMs. It works by creating and controlling isolated virtual environments.

* Virtual Machine is a software-based computer system. It is used to run operating systems independently inside another system. It works by using virtual hardware provided by the hypervisor.

* VirtualBox is a Type-2 hypervisor application. It is used to create and manage virtual machines on laptops/desktops. It works by running virtual hardware on top of the host operating system.

* ISO File is a disk image containing operating system files. It is used to install operating systems. It works by acting like a virtual installation disk.

* Snapshot is a saved state of a virtual machine. It is used to restore systems after errors or experiments. It works by storing VM memory and disk state.

* NAT Networking is a virtual networking mode. It is used to provide internet access to VMs through the host machine. It works by translating VM network traffic through the host IP.

* Bridged Adapter is a VM network configuration mode. It is used to give the VM its own network identity. It works by directly connecting the VM to the physical network.

* Root Directory is the top-level directory in Linux. It is used as the starting point of the filesystem. It works by containing all system folders under it.

* Home Directory is the personal workspace of a Linux user. It is used to store user files and settings. It works by providing isolated user-specific storage.

* Root User is the administrator account in Linux. It is used to perform unrestricted system operations. It works by having full permissions over the operating system.

* sudo is a Linux privilege escalation command. It is used to execute commands with administrator permissions. It works by temporarily granting elevated privileges.

* File Permissions is a Linux access control mechanism. It is used to restrict file access and operations. It works by defining read, write, and execute rights.

* Process is a running instance of a program. It is used to perform active tasks in the operating system. It works by consuming CPU and memory resources.

* Service is a background system process. It is used to provide continuous functionality like networking or databases. It works by running continuously in the background.

* Daemon is a Linux background service process. It is used to handle system or network requests continuously. It works by waiting silently for events or requests.

* systemctl is a Linux service management command. It is used to control and monitor services. It works by interacting with the systemd service manager.

* Package Manager is a software management tool in Linux. It is used to install, update, and remove applications. It works by managing software repositories and dependencies.

* apt is a Linux package management command. It is used mainly in Ubuntu/Debian systems to manage software. It works by downloading packages from repositories.

* yum is a Linux package management command. It is used mainly in CentOS and Red Hat systems. It works by resolving dependencies and installing packages.

* IP Address is a unique network identifier. It is used to identify devices on a network. It works by enabling communication between systems.

* Hostname is a human-readable system name. It is used to identify machines easily on networks. It works by mapping names to IP addresses.

* DNS is a domain name resolution system. It is used to convert domain names into IP addresses. It works by querying DNS servers for address mapping.

* Port is a logical communication endpoint. It is used to identify network services on a system. It works by routing traffic to specific applications.

* Ping is a network testing utility. It is used to check connectivity between systems. It works by sending ICMP echo requests and waiting for replies.

* SSH is a secure remote access protocol. It is used to connect to remote systems safely. It works by encrypting communication between client and server.

* OpenSSH is an SSH protocol implementation. It is used to provide secure remote access services. It works by running SSH server and client applications.

* Remote Access is the ability to control another system over a network. It is used for server management and support. It works through secure communication protocols like SSH.

* PuTTY is a remote terminal client for Windows. It is used to access Linux servers remotely. It works by establishing SSH or Telnet sessions.

* MobaXterm is an advanced remote terminal application. It is used to manage multiple remote server sessions. It works by integrating SSH, file transfer, and terminal tools.

* SCP is a secure file transfer protocol. It is used to transfer files between systems securely. It works by using SSH encryption during file transfer.

* SFTP is a secure file transfer protocol over SSH. It is used to transfer and manage files remotely. It works through encrypted SSH sessions.

* WinSCP is a graphical file transfer application for Windows. It is used to move files between Windows and Linux systems. It works using SCP or SFTP protocols.

* rsync is a file synchronization utility. It is used to efficiently copy and sync files between systems. It works by transferring only changed data blocks.

* Kali Linux is a security-focused Linux distribution. It is used for ethical hacking and penetration testing. It works by providing pre-installed cybersecurity tools.

* Penetration Testing is authorized security testing. It is used to identify vulnerabilities in systems. It works by simulating real-world cyber attacks.

* Ethical Hacking is legal security testing activity. It is used to improve system security. It works by identifying and reporting weaknesses responsibly.

* Encryption is a data protection mechanism. It is used to secure information during storage or transmission. It works by converting readable data into encoded format.

* Hashing is a one-way data transformation technique. It is used mainly for password storage and verification. It works by generating fixed-length hash values.

* Brute Force Attack is a password guessing attack method. It is used to gain unauthorized access. It works by trying many password combinations automatically.

* John the Ripper is a password auditing tool. It is used to test password strength and security. It works by attempting password cracking against hashes.

* Infrastructure is the foundational IT environment. It is used to host and run applications and services. It works through servers, storage, and networking systems.

* Server is a computing system providing services or resources. It is used to host applications, websites, or databases. It works by responding to client requests over networks.

* Cloud Computing is internet-based computing delivery. It is used to provide scalable computing resources on demand. It works through remote data centers managed by cloud providers.

* AWS is a cloud computing platform by Amazon. It is used for hosting, storage, networking, and DevOps services. It works by providing on-demand infrastructure services over the internet.

* Authentication is an identity verification process. It is used to validate users or systems before granting access. It works by checking credentials like passwords or keys.

* SSH Key is a cryptographic authentication method. It is used for secure passwordless SSH login. It works using public-key and private-key encryption.

* Logs is recorded system activity information. It is used for monitoring and troubleshooting systems. It works by storing event and error details generated by applications and services.

* Troubleshooting is a problem-solving process in IT systems. It is used to identify and fix technical issues. It works through systematic analysis and testing.
