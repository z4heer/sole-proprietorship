# 🧠 HOW TO USE THESE (IMPORTANT)

For every question:

* Speak like: *“Okay… first this happens… then…”*
* Don’t give textbook answers
* Always explain **flow**

# 🔰 LEVEL 1 — BASIC FLOW (ENTRY LEVEL)

---

👉 What interviewer expects:

* Frontend → API → Backend → DB → Response

---

* Request handling

* DB interaction
