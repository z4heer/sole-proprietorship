# 🚀 System Design Flow Learning

## 🧠 Purpose

This repository is designed to build **deep system understanding** for a **Senior Fullstack / Backend Engineer** role.

It focuses on:

* Flow-based learning (no page memorization)
* System thinking (end-to-end understanding)
* Verbal clarity (speaking while learning)
* Real-world engineering mindset

---

## ❌ Problems Addressed

* Visual Page Memory (remembering notes instead of concepts)
* Silent Processing (thinking without speaking)

---

## ✅ Learning Approach

Each topic is learned using:

1. **1-Liner** → Core idea
2. **Flow** → Step-by-step movement
3. **Sub-Flows** → Key components
4. **Keyword Expansion** → Performance, Security, Scale, Failure
5. **Speaking Practice** → Walk and explain

---

## 🔁 Daily Practice Method

1. Pick 1 topic (from folders)
2. Use AI prompt to generate flow
3. Close notes
4. Walk and explain out loud (no silence)
5. Record audio (optional)
6. Repeat next day without seeing

---

## 📂 Folder Structure

* `01_Foundations` → Core system basics
