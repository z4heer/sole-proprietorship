# 🛒 1. Product Browsing

**Definition:**
Product browsing allows users to view available products. It is used to explore items before purchase.

**How it works:**
🟢 User
First, user opens product page.
Then, frontend calls product API.
After that, backend fetches products from DB/cache.
Finally, products are displayed.

📦 Product API, UI, DB
💡 View product list

**Covered Features:**

* Product listing
* API integration
* UI rendering

**How it works:**
