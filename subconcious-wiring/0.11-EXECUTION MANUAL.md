# 🧠 UNIVERSAL SKILL EXECUTION MANUAL (START → AUTOMATION)

# 🧩 PHASE 1: SKILL SELECTION & BREAKDOWN

## 🔹 Step 1: Define Skill Clearly

❌ Wrong:

* “Learn trading”

✅ Correct:

* “Identify breakout + take entry with SL”

👉 Rule:
**Skill must be executable, not vague**

---

## 🔹 Step 2: Break Skill into Micro-Skills

Ask AI:

```
Break [skill] into:
- 5–10 sub-skills
- each should be actionable
```

### Example (Trading):

| Skill            | Micro Skills                         |
| ---------------- | ------------------------------------ |
| Breakout Trading | Trend, S/R, volume, entry timing, SL |

---

## 🔹 Step 3: Convert into “Observable Actions”

👉 Each micro-skill must answer:

* What do I SEE?
* What do I DO?

Example:

| Micro Skill | Observable Action    |
| ----------- | -------------------- |
| Trend       | Identify HH/HL       |
| Entry       | Buy above resistance |

---

# 📥 PHASE 2: DATA COLLECTION (VERY IMPORTANT)

## 🔹 Step 4: Get Raw Practice Data

### Use AI:

```
Generate 20 practice scenarios for [skill]
Include:
- easy to difficult
- real-world variation
```

---

### Sources by Domain:

#### 📈 Trading

* Charts (TradingView)
* Past market examples

#### 💻 Coding

* Problems (basic → advanced)
* Buggy code

#### 🗣️ Communication

* Interview questions
* Real-life situations

---

## 🔹 Step 5: Convert Data into Drills

👉 Each data point → 1 drill

Example:

| Data     | Drill                  |
| -------- | ---------------------- |
| Chart    | Identify trend + entry |
| Question | Answer in structure    |
| Problem  | Solve in code          |
