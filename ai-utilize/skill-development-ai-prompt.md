Yes — below is a reusable **AI prompt template** you can use for a similar **skill development project** for **any technical or non-technical skill**.

It is designed to help you build a **systematic learning, tracking, practice, review, and showcase framework** as a project, not just a study list.

## Master AI prompt

```text
I want to build a systematic skill development project for the following skill/domain:

[INSERT SKILL / DOMAIN NAME]

Purpose:
I want to learn this skill deeply, practice it systematically, track progress like a project, and build visible outcomes such as notes, diagrams, projects, recordings, presentations, portfolio items, or practical demonstrations.

Please act like an expert learning architect, curriculum designer, project planner, and skill coach.

Help me create a complete skill development system with the following:

1. Requirement summary
- Summarize my goal clearly
- Define what success looks like
- Mention beginner, intermediate, and advanced outcomes

2. Skill breakdown
- List all core skills
- List sub-skills under each core skill
- Remove duplicates
- Group them logically
- Separate must-have, good-to-have, and advanced skills

3. Related domains
- Include direct skills, support skills, indirect related skills, and behavioural skills
- Mention technical and non-technical support areas if relevant

4. Job roles or real-life use cases
- List job roles, business use cases, freelance use cases, personal use cases, or practical applications related to this skill
- Map each role/use case to the required skills

5. Physical or real-world awareness
- If relevant, include hardware, tools, devices, systems, materials, environments, or real-world components involved in this skill
- Help me understand what is physically or practically happening in the background

6. Behavioural skills
- Include Read, Write, Speak, Showcase
- Add problem solving, communication, explanation, documentation, presentation, teaching, observation, self-review, and feedback handling if relevant

7. Scientific learning methods
- Show how to practice using:
  - Feynman Technique
  - Active Recall
  - Spaced Repetition
  - Visualization
  - Deliberate Practice
  - Interleaving
  - Retrieval practice
  - Reflection and error logging
  - Audio/video self-recording
- Explain how to apply each method specifically for this skill

8. Analogies
- Give simple analogies for each core topic and major sub-skill
- Make the analogies practical and memorable

9. Learning roadmap
- Create a phased roadmap:
  - Foundation
  - Core practice
  - Applied practice
  - Real-world simulation
  - Showcase
  - Review and interview/readiness
- Mention dependencies and recommended order

10. Practice system
- Suggest daily, weekly, and monthly practice routines
- Include extensive reading, writing, speaking, observing, doing, and reviewing
- Suggest mini exercises, drills, simulations, mock sessions, and full projects

11. Output-based learning
- For each major skill, suggest outputs such as:
  - Notes
  - Cheat sheets
  - Mind maps
  - Architecture diagrams
  - Flowcharts
  - Case studies
  - Worksheets
  - Practice logs
  - Audio recordings
  - Video recordings
  - Presentations
  - Mini projects
  - End-to-end projects
  - Portfolio artifacts
  - Teaching materials
- Focus on building visible proof of learning

12. Tracking system
- Create a skill matrix table with columns such as:
  - Domain
  - Core Skill
  - Sub-skill
  - Related Role/Use Case
  - Tools/Hardware/Environment involved
  - Learning Method
  - Output to Build
  - Current Level
  - Target Level
  - Status
  - Review Cycle
  - Evidence
  - Notes
- Prefill as much as possible

13. Progress levels
- Define measurable skill levels such as:
  0 = No exposure
  1 = Awareness
  2 = Can follow guided practice
  3 = Can perform independently
  4 = Can explain, troubleshoot, and teach

14. Review and retention system
- Create revision cycles like 1 day, 7 days, 21 days, monthly
- Suggest what to review and how to test retention

15. Project management approach
- Treat this as a project
- Suggest milestones, phases, backlog, weekly plan, dashboard metrics, and review checkpoints
- Include risk factors, common failure points, and mitigation actions

16. Reference resources
- Suggest the best types of tutorials, books, practice resources, communities, documentation, videos, mentors, and project sources
- Mention what kind of source is best for each phase

17. Deliverables
- Give final output in these formats:
  a. Requirement summary
  b. Unique master skill list
  c. Skill matrix table
  d. 30-60-90 day roadmap
  e. Daily/weekly practice plan
  f. Project ideas
  g. Review system
  h. Portfolio/showcase plan

Important instructions:
- Keep it structured and practical
- Avoid duplicate skills
- Focus on deep understanding, practice, retention, and visible outcomes
- Prefer project-based learning over passive reading
- Use simple language
- If the skill is non-technical, still include behavioural, practical, and outcome-based learning systems
- If the skill is technical, include hardware/tools/software/environment awareness where relevant
- Where useful, include tables
- Where useful, include examples
- Help me build a long-term repeatable system, not just a one-time study plan
```

## Short version

If you want a shorter reusable prompt, use this:

```text
Help me build a complete skill development project for [SKILL NAME].

I want:
- requirement summary
- unique core skills and sub-skills
- related job roles or practical use cases
- direct, support, indirect, and behavioural skills
- scientific learning methods like Feynman, active recall, spaced repetition, visualization, deliberate practice
- analogies for major topics
- daily/weekly/monthly practice system
- output-based learning: notes, diagrams, recordings, projects, portfolio artifacts
- skill matrix table with tracking columns
- 30-60-90 day roadmap
- review and retention system
- project ideas from beginner to advanced
- portfolio/showcase plan

Treat this as a long-term project, not a study list. Avoid duplicates and organize everything systematically.
```
